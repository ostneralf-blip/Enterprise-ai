Wie sollen Komponenten im Diagramm ausgewählt werden?

Jede Variante bestimmt, wie Nutzer ihre Architektur zusammenstellen

A

Checkbox-Auswahl (Multi-Select)

Jede Komponente hat eine Checkbox. Mehrere können gleichzeitig aktiviert sein. Rechte Seite zeigt "Meine Architektur" als Zusammenfassung der Auswahl — inkl. Inkompatibilitäts-Warnungen und Vorschläge.

Vorschau — Checkbox-Modus
SAP Datasphere DSGVO ✓
SAP AI Core DSGVO ~
⚠️ vLLM ist inkompatibel mit SAP AI Core (konkurrierende Inference-Schicht)
B

Klick = Detail + Highlight (Single-Focus)

Klick auf eine Komponente öffnet das Detail-Panel (wie heute). Zusätzlich werden alle suggests-Komponenten grün hervorgehoben, incompatible_with-Komponenten rot markiert. Keine persistente Multi-Auswahl.

Vorschau — Focus-Modus
SAP AI Core ← fokussiert SAP HANA Cloud ✓ empfohlen vLLM ✗ inkompatibel Andere neutral
C

Kombiniert: Auswahl + Focus

Checkbox zum Einschließen (= "Ich nutze das"), Klick auf den Namen öffnet Details und hebt Beziehungen hervor. Beide Gesten gleichzeitig nutzbar. Am mächtigsten, aber auch am komplexesten zu implementieren und zu verstehen.

Vorschau — Kombiniert
SAP AI Core ← Detail-Klick Checkbox = ausgewählt, Klick = Details
vLLM ⚠️ Konflikt mit SAP AI Core
