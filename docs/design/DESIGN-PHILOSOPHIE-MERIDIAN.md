# MERIDIAN
### Eine Designphilosophie

---

**Meridian** ist die Ästhetik der stillen Vermessung. Sie glaubt, dass Klarheit nicht durch Lautstärke entsteht, sondern durch Präzision: eine Haarlinie, exakt gesetzt; eine Zahl, die atmen darf; eine Fläche aus warmem Elfenbein, auf der jedes Element seinen vermessenen Ort hat. Wie auf einer Seekarte des 19. Jahrhunderts trägt jede Linie Bedeutung — nichts ist Dekoration, alles ist Koordinate. Das Werk wirkt, als hätte ein Kartograf Wochen über seinem Tisch verbracht: unzählige Stunden des Justierens, Verwerfens, Neuvermessens, bis jede Beziehung zwischen den Elementen stimmt. Diese Sorgfalt ist nicht Stil, sie ist Haltung.

Der Raum ist das erste Material. Meridian arbeitet mit großzügigen Rändern wie ein gebundenes Buch mit Vorsatzpapier — der leere Raum ist kein Rest, sondern das Fundament, auf dem Information ruht. Kompositionen bauen auf einem strengen, unsichtbaren Raster, das nie gezeigt, aber immer gespürt wird. Elemente stehen nie zufällig nebeneinander; sie sind zueinander vermessen wie Längengrade. Dichte entsteht nur dort, wo Daten sprechen — und selbst dort bleibt zwischen jeder Ziffer, jedem Balken, jeder Markierung ein disziplinierter Abstand, der von meisterhafter Beherrschung zeugt.

Farbe ist Sprache, nicht Schmuck. Ein einziges tiefes Blau — das Blau technischer Tinte auf Karten — trägt alle Betonung: Es markiert das Wesentliche und schweigt sonst. Es ruht auf Elfenbein, wird flankiert von einer Skala aus Schiefergrau, das vom fast schwarzen Ink bis zur zartesten Trennlinie reicht. Punktuell dürfen semantische Töne auftreten — ein gedämpftes Smaragd, ein warmes Bernstein, ein leises Violett — aber stets als kleine, präzise gesetzte Signale, nie als Fläche. Die Palette ist so eng, dass jede Farbentscheidung wie das Ergebnis langer Kalibrierung wirkt — weil sie es ist.

Typografie folgt der Ordnung eines wissenschaftlichen Tagebuchs: Eine Serifenschrift von literarischer Wärme spricht in großen, ruhigen Zahlen und knappen Überschriften — nie in Textblöcken. Eine technische Monospace flüstert Referenzmarken, Koordinaten, Datumszeilen in Kapitälchen-Disziplin. Eine sachliche Grotesk trägt die wenigen notwendigen Worte. Text ist Akzent, nie Erklärung: Was gezeigt werden kann, wird gezeigt — durch Balken mit Messstrichen, Ringe, gesetzte Punkte im Koordinatenfeld, Verlaufslinien. Eine einzelne große Ziffer sagt mehr als ein Absatz; ihre Platzierung ist das Ergebnis peinlich genauer Abwägung.

Meridian dokumentiert das Unsichtbare mit der Ehrfurcht eines Naturforschers: Systematische Wiederholung — Messstriche, Punktreihen, feine Skalen — erzeugt Tiefe und belohnt das zweite Hinsehen. Jede Seite wirkt wie das Blatt einer erfundenen Disziplin, deren Instrumente man sofort versteht: Hier wurde etwas Flüchtiges vermessen, kartiert, festgehalten. Nichts überlappt, nichts fällt aus dem Satzspiegel, jede Linie endet, wo sie enden soll. Das Ergebnis muss aussehen wie das Werk eines Meisters seines Fachs — ein Artefakt von solcher handwerklicher Dichte, dass man es aufheben, rahmen und einem Vorstand vorlegen möchte.
